# Formulário com validação de input

<p><b>Esta é uma solução para o desafio do curso DevQuest.</b></p>

## Visão Geral 

###  Projeto 
<b>Este é um projeto que faz parte de um desafio do curso DevQuest. O objetivo é criar uma página de formulário com validação de input, desenvolvendo habilidades de HTML, CSS e JavaScript.

###  Desafio

<b>O desafio consiste em construir uma página de formulário a partir dos designs e funcionalidades fornecidas. 
Ao clicar no botão para enviar o formulário, a borda dos inputs altera de cor, sendo verde se todos os inputs estiverem preenchedos. E caso algum dos inputs não estiver preenchido, altera a borda para vermelha e uma mensagem de "campo obrigatório" deve aparecer embaixo do input.

### Funcionalidades 
<ul>
<li>Ao clicar no botão para enviar o formulário, dependendo do valor do imput, a cor da borda altera.
</ul>
